# Demo Video Guidelines

If you record a demo video for this project, include the following sections:

1. Introduction
   - Describe the project and its purpose.
   - Mention the technologies used: Python, Streamlit, MySQL, Ollama/GPT-OSS.

2. Setup Overview
   - Install dependencies from `requirements.txt`.
   - Configure the `.env` file.
   - Start MySQL and Ollama.
   - Launch the Streamlit app.

3. Feature Walkthrough
   - Register a new user.
   - Log in and start a conversation.
   - Send chat messages and show assistant responses.
   - Show conversation history saved in the database.

4. Backend / Database
   - Explain the database schema in `db/schema.sql`.
   - Describe how users, conversations, and messages are stored.

5. Closing
   - Summarize the project capabilities.
   - Mention how to run the app locally.

> Keep the demo concise and focused on the main user experience.
